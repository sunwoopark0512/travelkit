---

## 1️⃣ 오늘 해야 할 3가지 (30–60분)

**A. 판매 페이지 배포**
*   `docs/SALES_PAGE_DELUXE.md` → 노션/카페24/티스토리 중 하나에 **그대로 게시**
*   상단 CTA 버튼 1개만 유지: **[Deluxe 설치 신청하기]**

**B. 결제 연결**
*   결제수단 1개만 선택(토스페이먼츠/스마트스토어/카카오페이 중 택1)
*   상품명: **Silver Creator OS · Deluxe (설치대행 + 1회 성과보장)**
*   가격: **149,000원**

**C. 결제 후 자동 안내**
*   `docs/POST_PURCHASE_MESSAGES.md` → 문자/카톡 자동발송 세팅
*   포인트: **설문 링크 + 설치 예약 링크**만 확실히 전달

---

## 2️⃣ 첫 고객 처리 플로우 (OPS 그대로 읽기)

1.  **결제 확인** → 자동 메시지 발송
2.  **사전 설문 수집** → 설문 답을 **INBOX 템플릿**에 복붙
3.  **원격 설치** → `DELUXE_EXECUTION_KIT.md` 30분 스크립트 **그대로 낭독**
4.  **Aha Moment** → `SUCCESS: wrote SCRIPTS` 로그 **눈앞에서 확인**
5.  **파일 전달** → 생성된 30분 대본 JSON + 간단 사용 가이드

> 실패 시 규칙: **10분 내 결과 미생성 = 전액 환불** (신뢰 극대화)

---

## 3️⃣ 내일 할 일 (매출 가속)

*   **지인 1명**에게만 제안(설명 금지, 페이지 링크만 전달)
*   DM 문구(한 줄):
