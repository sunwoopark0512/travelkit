# Reference: Stitch Hotkeys & Shortcuts

## Procedures

### Core Navigation
- **Select Mode**: `V`
- **Pan Mode**: `H` (or hold `Space`)
- **Zoom Mode**: `Z` (Click to in, Alt+Click to out)
- **Fit to View**: `0` or `Cmd+1`
- **Command Palette**: `Cmd+K` / `Ctrl+K`

### Screen Management
- **Next Screen**: `Cmd+Right` / `Ctrl+Right`
- **Prev Screen**: `Cmd+Left` / `Ctrl+Left`
- **Duplicate**: `Cmd+D` / `Ctrl+D`
- **Download**: `Shift+D`

## Input / Output Examples
**Input**: Press `Cmd+K` -> Type "New screen"
**Output**: A new screen is generated on the canvas.

**Input**: Select screen -> `Cmd+D`
**Output**: Exact clone of the screen appears next to the original.

## Gold Questions
Q: What is the shortcut to open the universal Command Palette?
A: `Cmd+K` (Mac) or `Ctrl+K` (Windows).

Q: How do you quickly pan the canvas while in Select mode?
A: Hold `Space` bar.

Q: What is the fastest way to find the list of all shortcuts?
A: Press `?`.
