    Deluxe Plan Sales Page (1-Page Copy)¶

    [Title] "컴맹이어도 상관없습니다. 컴퓨터 켜기만 하세요." 당신의 PC에 '유튜브 자동수익 비서'를 직접 설치해 드립니다. (149,000원 / 선착순 10명 한정)

    1. 당신이 정말 두려워하는 것¶

    유튜브가 돈이 된다는 건 다들 압니다. 그런데 왜 시작을 못 할까요? * "내가 프로그램을 깔 줄 몰라서..." * "복잡한 API 키? 코딩? 듣기만 해도 머리 아파서..." * "비싼 강의 들었는데, 정작 내 컴퓨터에선 안 돼서..."

    당신 잘못이 아닙니다. '기술'이라는 벽이 너무 높았을 뿐입니다.

    2. 우리가 대신 부숴드립니다 (Deluxe Plan)¶

    강의 듣고 따라 하는 게 아닙니다. 저희가 직접 당신의 컴퓨터에 들어가서 다 해드립니다.

    • 원격 설치 대행: 복잡한 환경 설정, API 연동, 폴더 정리... 전문가가 1부터 100까지 다 해드립니다.
    • 1회 성과 보장 (One-Done): 그냥 설치만 하고 "끝났습니다" 하지 않습니다. 당신의 첫 번째 30분 대본이 완성되는 꼴을 볼 때까지 접속을 끊지 않습니다.

    3. 당신이 가져올 건 딱 하나¶

    "살면서 겪은 이야기꾸러미" * 주식으로 돈 날린 사연 * 며느리한테 서운했던 일 * 젊은 날의 후회

    이것만 엑셀에 한 줄 적으세요. 나머지는 AI 비서가 알아서 기승전결(30분 분량)로 뽑아냅니다.

    4. 가격 및 구성¶

    Deluxe Plan: 149,000원 (정가 490,000원)

    1. Silver Creator OS 영구 라이선스 (평생 소장)
    2. 1:1 원격 설치 서비스 (30분 소요)
    3. 첫 대본 제작 컨설팅 (소재 선정부터 출력까지)
    4. [특전] 평생 AS 기술지원 단톡방 초대

    5. 미친 환불 보장 (Risk Reversal)¶

    "설치 후 10분 안에 첫 대본이 안 나오면, 묻지도 따지지도 않고 100% 전액 환불해 드립니다." 당신이 잃을 건 0원입니다. 하지만 얻을 건 '인생 2막'입니다.

    [지금 바로 신청하기] (버튼: 설치 일정 예약하기) 현재 대기 인원이 많아 설치가 지연될 수 있습니다.
